Gestió de les galetes

Aquest document explica què són les galetes, per a què s'utilitzen, i com podeu controlar les galetes que teniu emmagatzemades al vostre disc dur amb el Gestor de galetes del &brandFullName;.

En aquesta secció:
  • Què és una galeta?
  • Establir normes per a les galetes
  • Acceptar i blocar les galetes
  • La finestra d'excepcions de les galetes
  • La finestra de visualització de les galetes

Què és una galeta?

Una galeta és un fitxer creat per un lloc web que s'emmagatzema al vostre ordinador i que conté certa informació, com ara les vostres preferències sobre aquella web. Quan visiteu un lloc que fa servir galetes, el lloc demana permís al &brandShortName; per desar una o diverses galetes al vostre disc dur.

Després, quan torneu a aquest mateix lloc, el &brandShortName; torna les galetes que anteriorment havia desat. Això permet veure pàgines personalitzades, adequades a les vostres preferències.

Les galetes també poden contenir informació que permeti identificar-vos, tal com el vostre nom, adreça electrònica, adreça postal, o número de telèfon. No obstant, heu de tenir en compte que qualsevol lloc web només té les dades que vosaltres mateixos li doneu. Per exemple, un lloc web no pot obtenir la vostre adreça electrònica si no és que vosaltres mateixos la faciliteu (e.g. l'escriviu en un formulari). Els llocs web tampoc tenen accés a cap altra informació desada al vostre ordinador.

En la configuració per defecte, tot aquest mecanisme és invisible de cara a l'usuari, i no sabreu si un lloc web crea una galeta o si el &brandShortName; torna una galeta al lloc web al qual pertany. No obstant, podeu configurar el &brandShortName; per a que us demani si accepteu cada una de les galetes.

Establir normes per a les galetes

Per defecte el &brandShortName; accepta totes les galetes. Si voleu tenir més control sobre les galetes que es desen al vostre ordinador, seguiu les següents instruccions:

  1. Seleccioneu &pref.menuPath; i obriu el quadre de Privadesa.
  2. Seleccioneu que tanqui el &brandShortName; des del menú Conserva-les fins.

Si voleu atorgar als llocs en què confieu la capacitat de poder emmagatzemar galetes de forma permanent (ex., permetre que entreu automàticament a un lloc), feu clic a Excepcions…, introduïu l'adreça del lloc, i feu clic a Permetre.

Paràmetres de les galetes

També hi ha altres &pref.plural; que afecten la gestió de les galetes del &brandShortName;.

Accepta galetes dels llocs web

Si no voleu que cap lloc emmagatzemi galetes al vostre ordinador, desmarqueu aquesta &pref.singular;. Tingueu en compte que algunes webs pot ser que no funcionin bé si teniu les galetes inhabilitades. Si voleu atorgar a alguns llocs la capacitat d'emmagatzemar galetes, obriu la finestra d'excepcions de les galetes fent clic a Excepcions…, introduïu l'adreça del lloc i feu clic a Permet o Permet per a aquesta sessió.

Conserva-les fins:
  • que vencin
    Si se selecciona aquesta &pref.singular;, se suprimirà la informació de la galeta quan aquesta arribi a la seua data de venciment.
  • que tanqui el &brandShortName;
    Si se selecciona aquesta &pref.singular;, la informació de la galeta se suprimirà quan reinicieu el &brandShortName;. Els llocs web que necessiten l'ús de galetes continuaran funcionant, però si reinicieu el &brandShortName;, el lloc web creurà que hi entreu un altre cop per primera vegada.
  • demana-m'ho cada vegada
    Mostra una avís cada vegada que que un lloc prova d'emmagatzemar una galeta, per a demanar-vos si la voleu emmagatzemar.

Acceptar i blocar les galetes

Si heu triat demana-m'ho cada vegada del menú Conserva-les fins, veureu un diàleg amb les següents opcions sempre que un lloc provi d'emmagatzemar una galeta al vostre ordinador:

Permet
Per a admetre aquesta galeta en particular, només cal que feu un clic a Permet. Utilitzeu aquesta opció si confieu en el lloc.
Permet per a aquesta sessió
Si voleu permetre la galeta per a la sessió actual, però que es suprimeixi quan sortiu del &brandShortName;, cliqueu Permet per aquesta sessió. Utilitzeu-ho si el lloc web no funciona sense les galetes però no voleu que les seues galetes s'emmagatzemin permanentment.
Denega
Si no voleu emmagatzemar la galeta, feu un clic a Denega. Utilitzeu-ho si no confieu en el lloc o sospiteu que pot comprometre la vostra privadesa.
Utilitza la meua elecció per a totes les galetes d'aquest lloc
Seleccioneu aquest quadre de verificació abans de clicar qualsevol dels botons si voleu que el &brandShortName; recordi la vostra decisió i no ho torni a demanar. El lloc s'afegirà a la finestra d' excepcions de galetes, on podeu modificar la vostra elecció sempre que vulgueu més endavant.

La finestra d'excepcions de les galetes

Accediu a aquesta finestra clicant al botó Excepcions… a les &pref.plural; de les galetes. Aquí podeu definir excepcions a les vostres &pref.plural; generals per a les galetes de llocs específics. Utilitzant les excepcions, podeu permetre totes les galetes, rebutjar-les totes, o permetre-les però suprimir-les quan sortiu del &brandShortName;, independentment d'altres opcions que tingueu definides.

Per a afegir un lloc a la llista, simplement introduïu el nom de domini del lloc al camp Adreça del lloc web. Llavors feu clic a Bloca per a rebutjar les galetes del lloc, feu clic a Permet per a permetre-les o cliqueu a Permet per a aquesta sessió per a permetre-les des d'aquest lloc però suprimir-les quan sortiu del &brandShortName;.

Per a treure un lloc de la llista, seleccioneu-lo i feu clic a Suprimeix el lloc. Per a esborrar completament tota la llista, feu clic a Suprimeix tots els llocs. Això eliminarà totes les normes establertes i s'aplicaran les &pref.plural; generals per a les galetes.

La finestra de visualització de les galetes

Utilitzeu la finestra Visualitza les galetes per a veure o eliminar les galetes. Per a accedir-hi, cliqueu al botó Visualitza les galetes a les &pref.plural; de les galetes.

Aquesta finestra mostra una llista de totes les galetes que actualment teniu emmagatzemades al vostre ordinador. Les podeu seleccionar per a veure informació addicional sobre cada una d'elles. Podeu també cercar un lloc o un nom d'una galeta escrivint-ho a la barra de cerca.

Per a eliminar una galeta de la llista, seleccioneu-la i feu clic a Suprimeix la galeta. Per a eliminar totes les galetes, feu clic a Suprimeix totes les galetes.
